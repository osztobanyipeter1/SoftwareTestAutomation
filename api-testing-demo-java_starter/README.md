# API Testing Demo (Java 21, Gradle, JUnit 5, RestAssured)

A minimal Java project for teaching API testing.

## Prerequisites
- Java 21
- Gradle or Gradle Wrapper

## Usage
```bash
gradle wrapper --gradle-version 8.10.2
./gradlew test
./gradlew test -Denv=qa
```
Reports:
- build/reports/tests/test/index.html
